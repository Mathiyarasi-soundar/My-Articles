# My Articles Project

This project allows users to add, view, and manage articles on a simple webpage. The core functionality is implemented using HTML for structure, CSS for styling, and JavaScript for interactivity and dynamic content handling.

## Project Overview

The "My Articles" project is a web application that enables users to add new articles with titles, author name and content. The articles are displayed dynamically on the page, and the user can add multiple articles, which will persist as long as the session lasts.


## Features

- Add a new article with a title and content.
- Display added articles in a list.
- Basic form validation for input fields (e.g., ensuring title and content are not empty).
- Clear form inputs after article submission.
- Simple, user-friendly design for easy interaction.

## Technologies Used

- **HTML**: For structuring the webpage and the form.
- **CSS**: For styling the form and articles, including responsive design features.
- **JavaScript**: For handling user interaction (e.g., adding articles, form validation, etc.).

## Usage
-- Open the index.html file in your browser.
-- You will touch the plus symbol to add article title and content.
-- Enter a title, author name and some content for the article.
-- Click the "Add" button to submit the article.
-- The article will appear below the form in a list format, displaying the title, author name and content of each article.
-- Repeat the process to add more articles.
